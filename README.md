# Приложение прогноз погоды на TypeScript

## Описание
Приложение для отображения погоды с использованием внешнего API. Реализована асинхронная загрузка данных с применением async/await и TypeScript.

### Технологии:
TypeScript, REST API

### Как запустить
Проект можно запустить по этой ссылке: https://nicomsk.github.io/Weather-forecast-app/
