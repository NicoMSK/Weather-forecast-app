<h1>Приложение прогноз погоды на TypeScript</h>

<h2>Описание</h2>
<p>Веб-приложение для отображения прогноза погоды с возможностью выбора города. Пользователь может просматривать прогноз с шагом в 2 часа на текущий день, прогноз на завтра и на ближайшие 3 дня.

Данные загружаются из внешнего погодного API с использованием асинхронных запросов (async/await). Приложение реализовано на TypeScript, с акцентом на типизацию данных, обработку состояний загрузки и обновление интерфейса при смене города.</p>

<h3>Технологии:</h3>
TypeScript, REST API, Vite
